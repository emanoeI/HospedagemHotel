<h1>🏨 Sistema de Reservas - Desafio Bootcamp DIO WEX - End to End Engineering</h1>

<p>Esse projeto nasceu como desafio do Bootcamp <strong>DIO WEX - End to End Engineering</strong>, usando <strong>C#</strong> como linguagem principal. 💻</p>

<h2>📋 O que é o projeto?</h2>

<p>Um sistema simples e prático para gerenciar reservas em hotel, com funcionalidades que incluem:</p>

<ul>
  <li>👤 Cadastro e controle dos hóspedes</li>
  <li>🛌 Definição e gerenciamento das suítes, respeitando capacidade para adultos e crianças</li>
  <li>📆 Criar reservas, calcular valores totais e aplicar descontos progressivos</li>
  <li>❌ Possibilidade de cancelar reservas</li>
</ul>

<h2>🎯 Por que fiz esse projeto?</h2>

<p>Para colocar em prática e fortalecer conceitos de:</p>

<ul>
  <li>🧠 Lógica de programação</li>
  <li>📚 Programação orientada a objetos (POO)</li>
  <li>✅ Boas práticas de código em C#</li>
  <li>💻 Interação via console</li>
</ul>

<h2>🔚 Considerações finais</h2>

<p>Esse projeto faz parte do meu portfólio e representa minha evolução técnica durante o bootcamp. Também foi uma ótima oportunidade para praticar Git e GitHub, garantindo um código versionado e organizado. 🚀</p>
